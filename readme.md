A premium, high-fidelity personal portfolio featuring immersive **Three.js** 3D visuals, studio-grade lighting, and cinematic "glassmorphism" user interfaces. Designed for impact and performance.

---

## 📽️ Visual Experience

- **Cinematic 3D Hero**: A custom-engineered dynamic Icosahedron centerpiece with studio RIM lighting and mouse-responsive parallax.
- **Glassmorphism 2.0**: Frosted glass UI elements using advanced `backdrop-filter` and semi-transparent borders for a futuristic depth.
- **Interactive Orbs**: Redesigned navigation elements (Scroll-to-Top and Live Chat) as glowing, spinning 3D orbs.
- **Micro-Animations**: Elastic bounce effects, gradient flows, and liquid-smooth scrolling for a high-end feel.

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla JavaScript (ES6+), CSS3
- **Graphics**: [Three.js](https://threejs.org/) (Custom Shader Geometry)
- **Styling**: Vanilla CSS (Custom Variable Design System)
- **Features**: Netlify Deployment, Google AdSense, EmailJS Integration, Tawk.to Live Chat (Custom Interface)

## 🎨 Design Philosophy

Every pixel is crafted to provide a "Cinematic" experience, drawing inspiration from high-budget Sci-Fi interfaces and premium studio branding. The use of a dark, high-contrast palette with **Cyan** and **Violet** secondary accents ensures maximum visual punch.
